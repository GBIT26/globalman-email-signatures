Setup Instructions

    
Gmail

    
    
      
  1. Click Copy HTML on your chosen design
    2. 
      
  2. Open Gmail → Settings (gear) → See all settings
    3. 
      
  3. Scroll to Signature → Create new
    4. 
      
  4. Click the editor, press Ctrl+A then Ctrl+V (Mac: Cmd+V)
    5. 
      
  5. Scroll down → Save Changes
    6. 
    

    
Outlook (Desktop & Web)

    
    
      
  1. Click Copy HTML
    2. 
      
  2. Outlook → Settings → Mail → Compose and reply
    3. 
      
  3. Under Email signature, click New signature
    4. 
      
  4. Paste with Ctrl+V / Cmd+V
    5. 
      
  5. Set as default → Save
    6. 
    

    
Apple Mail

    
    
      
  1. Click Copy HTML
    2. 
      
  2. Apple Mail → Settings → Signatures
    3. 
      
  3. Click + to create new
    4. 
      
  4. Paste with Cmd+V
    5. 
      
  5. Assign to your account
    6. 
    

    
Thunderbird

    
    
      
  1. Click Show HTML, then copy all code
    2. 
      
  2. Thunderbird → Account Settings
    3. 
      
  3. Check Use HTML and paste into the signature box
    4. 
      
  4. Click OK
    5.
